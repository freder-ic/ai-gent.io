# ai-gent

AI agents you can trust with your data.

ai-gent is an open ecosystem for privacy-first, resource-efficient AI agents —
for individuals, developers, and organizations with sensitive data.

## What is ai-gent?
- Build and run AI agents under your control
- Prefer the smallest viable models
- Open rules for governance and monetization
