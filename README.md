# ai-gent

ai-gent is an open ecosystem for building, sharing, and running AI agents.

The project focuses on:
- composable and reusable AI agents
- transparent governance and fair monetization
- long-term sustainability over short-term growth

## Project structure

- **ai-gent.io**  
  Entry point, vision, and orientation

- **ai-gent.store**  
  Curated marketplace for AI agents  
  (governance and publication rules already defined)

- **ai-gent.org**  
  Community, discussions, and collaborative governance  
  (coming next)

ai-gent is intentionally built in the open.
Rules, decisions, and evolution happen transparently
and with a strong bias toward trust and fairness.
